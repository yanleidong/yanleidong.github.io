SAP 工厂后台配置

SAP 工厂后台配置

创建工厂

事务代码：OX10

  1. 定义工厂
  2. 创建新条目
  3. 维护工厂数据信息
  4. 补充维护 工厂地址信息
  5. 保存

定义产品组

事务代码：OVXB

  1. 定义产品组
  2. 创建新条目
  3. 输入产品组和产品组名称
  4. 保存

定义销售组织

事务代码：OVX5

  1. 定义销售组织
  2. 创建新条目
  3. 维护销售组织数据和地址信息
  4. 保存

定义分销渠道

事务代码：OVXI

  1. 定义分销渠道
  2. 创建新条目
  3. 维护分销渠道和名称
  4. 保存

定义采购组织

事务代码：OX08

  1. 创建新条目
  2. 维护采购组织和描述
  3. 保存

定义仓库地点

事务代码：OX09

  1. 选择 对应的 工厂号
  2. 创建新条目
  3. 增加库存地点编号
  4. 保存

定义起运点

事务代码：OVXD

  1. 定义起运点
  2. 创建新条目
  3. 维护运送点信息和地址信息
  4. 保存

维护装载点

事务代码：OVX7

  1. 选择 装运点
  2. 创建新条目
  3. 维护装载点+责任人+描述
  4. 保存

工厂维护计划点

事务代码：OIX0

  1. 创建新条目
  2. 输入计划工厂
  3. 保存

给公司代码分配工厂

事务代码：OX18

  1. 创建新条目
  2. 输入公司代码+工厂
  3. 保存

将业务范围分配到工厂/评估范围和部门

事务代码：OMJ7

  1. 选择 工厂 - 部门
  2. 创建新条目
  3. 维护 工厂 + 产品组 + 部门
  4. 保存
  5. 选择 评估范围 - 部门
  6. 创建新条目
  7. 维护 评估范围 + 产品组 + 部门
  8. 保存

销售工厂/评估范围 - 产品组 - 业务范围数据比较

事务代码：OMJM

  1. 执行下就行

给公司代码分配销售组织

事务代码：OVX3

  1. 将缺失的公司代码补充完整
  2. 保存

给销售组织分配分销渠道

事务代码：OVXK

  1. 创建新条目
  2. 维护 销售组织 与 分销渠道 对应关系
  3. 保存

给销售组织分配部门

事务代码：OVXA

  1. 创建新条目
  2. 维护 销售组织 与 产品组 对应关系
  3. 保存

设置销售范围

事务代码：OVXG

  1. 创建新条目
  2. 维护 销售组织 + 分销渠道 + 产品组 对应关系
  3. 保存

给销售范围分配销售办公室

事务代码：OVXM

  1. 创建新条目
  2. 维护 销售组织 + 分销渠道 + 产品组 + 办公室
  3. 保存

给销售办公室分配销售组

事务代码：OVXJ

  1. 创建新条目
  2. 维护销售办公室 + 销售组对应关系
  3. 保存

分配销售组织 - 分销渠道 - 工厂

事务代码：OVX6

  1. 创建新条目
  2. 维护销售组织 + 分销渠道 + 工厂 对应关系
  3. 保存

给公司代码分配采购组织

事务代码：OX01

  1. 将公司代码补充完整
  2. 保存

给工厂分配采购组织

事务代码：OX17

  1. 创建新条目
  2. 维护采购组织 + 工厂 对应关系
  3. 保存

给工厂分配缺省的采购组织

事务代码：OMKI

  1. 将采购组织补充完整
  2. 保存

给工厂/存储地点分配仓库编码

  1. 创建新条目
  2. 维护 工厂 + 库位 + 仓库编码 对应关系
  3. 保存

给工厂分配起运点

事务代码：OVXC

  1. 选中 工厂， 点击 分配
  2. 选中 对应的 装运点
  3. 保存
