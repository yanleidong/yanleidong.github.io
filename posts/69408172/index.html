SAP 资产后台配置

固定资产组织结构

分配非税购置的进项税标识符

在此步骤中，您为每个公司代码指定一个进项税标识。 然后，当您过账不需要纳税但过账到与税收相关的科目的购置时，系统会使用此标识。

  1. 事务代码：
  2. 找到公司代码，维护销项税和进项税
  3. 保存

查看资产会计：国家信息

  1. 事务代码：
  2. 双击中国，查看设置的最低的固定资产金额

复制参考折旧表/折旧范围表

  1. 事务代码：OAP1
  2. 选择参考折旧表，然后我们进行复制
  3. 复制成功，返回
  4. 双击指定折旧表的说明

    5.保存

将折旧表分配至公司代码

  1. 事务代码：OAB1
  2. 找到自己的公司代码，将折旧范围维护上去
  3. 保存

创建屏幕格式规则

  1. 事务代码：
  2. 这边不可创建新条目，我们可以拷贝系统原始的1000-不动产，来创建新条目
  3. 保存

定义号码范围间隔

  1. 事务代码：AS08
  2. 输入公司代码，维护间隔
  3. 维护资产号段
  4. 保存

确定账户确定

  1. 事务代码：ORG2
  2. 创建新条目，维护科目
  3. 保存

定义资产分类

  1. 事务代码：AM01
  2. 创建新条目，维护数据

  3. 保存

定义资产主数据的屏幕格式

  1. 事务代码：OA77
  2. 选择定义资产主数据的屏幕布局
  3. 选择自己的资产类别，双击逻辑字段组，配置每个屏幕上的字段
  4. 保存

定义折旧范围过帐到总帐的方式

  1. 事务代码：
  2. 选择自己的折旧表
  3. 双击 1 帐面折旧
  4. 保存

分配总帐科目

  1. 事务代码：
  2. 选择 自己的 折旧表，双击 科目确定
  3. 选中 科目分配， 双击 资产负债科目
  4. 选中 科目分配， 双击 折旧
  5. 保存

定义终止值代码

  1. 事务代码：
  2. 创建新条目
  3. 维护终止值名称
  4. 保存
  5. 维护等级
  6. 保存

维护期间控制方法

  1. 事务代码：
  2. 创建新条目
  3. 保存

维护折旧码

  1. 事务代码：
  2. 创建新条目，
  3. 双击 计算方法的分配， 创建新条目
  4. 最后保存需要激活折旧码
  5. 按上面的步骤新建其他的折旧码

将资产分类对应的 折旧范围打开

  1. 事务代码：OAYZ
  2. 选中资产类， 点击折旧范围， 维护 折旧码和屏幕格式
  3. 保存， 其他的资产类也一样操作

锁定公司代码下的资产分类

这样我们在前台就看不到我们不需要的资产分类

  1. 事务代码：AM05
  2. 选中公司代码，将不需要显示的资产分类勾选上
  3. 保存
